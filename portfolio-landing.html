Sistema ERP para Empresas

// sobre mí

Código que resuelve
problemas reales.

Soy desarrollador backend y web con base en Argentina. Me especializo en construir sistemas que escalan — desde APIs robustas hasta interfaces que convierten.

No escribo código por escribir. Cada decisión técnica tiene un porqué de negocio detrás: performance, mantenibilidad, o conversión.

Trabajo remoto con clientes en Latinoamérica, Europa y Estados Unidos. Comunicación clara, entregas a tiempo, sin sorpresas.

01

Primero entiendo el problema

Antes de escribir una línea, entiendo qué necesita tu negocio. La solución técnica viene después.

02

Código limpio, no código heroico

El mejor código es el que otra persona puede leer, mantener y escalar. Sin magia innecesaria.

03

Entrego. Siempre.

Los deadlines existen. Comunico avances, gestiono expectativas y cumplo lo acordado.
